# Wiki backend

## Настройка

```
npm install
npm run typeorm:cli -- migration:run
```

## Запуск в окружении development

```
npm run serve
```

## Запуск в окружении production

```
npm run build
npm run start
```

## Документация API

Собрать документацию:

```
npm run apidoc
```

Путь к документации: `public/index.html`
